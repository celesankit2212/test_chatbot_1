 
# AI-Powered Test Automation Bot (Open Source)

## Overview
This project generates and executes automated test scripts using AI models from Open AI.

## Installation
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Add the required OpenAI API key:
   ```
   ADD THE 'OPEN_AI_KEY' in env variable to get the model work
   ```
3. Run the Flask app:
   ```bash
   python app.py
   ```

## Usage
- Enter a test case description in the UI and click "Generate & Run Test".
