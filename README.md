 
# AI-Powered Test Automation Bot (Open Source)

## Overview
This project generates and executes automated test scripts using AI models from Hugging Face.

## Installation
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Run the Flask app:
   ```bash
   python app.py
   ```

## Configuration
- Modify `config.py` to specify the Hugging Face model.

## Usage
- Enter a test case description in the UI and click "Generate & Run Test".
